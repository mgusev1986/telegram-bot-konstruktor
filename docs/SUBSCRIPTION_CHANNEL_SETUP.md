# Подписка + канал/чат: настройка

## 1. Секция под оплатой (уже есть)

**Как сделать секцию доступной только после оплаты:**

1. Backoffice → Бот → **Открыть** (или Платная секция)
2. **Создать продукт** — укажите цену, тип биллинга:
   - **ONE_TIME** — разовая оплата, доступ навсегда
   - **TEMPORARY** — доступ на N дней (укажите `durationDays`, например 30 для месяца)
3. В списке пунктов меню нажмите **«Ограничить»** у нужной секции
4. Выберите продукт → сохраните

После этого секция в боте будет открыта только пользователям с активной подпиской на этот продукт.

---

## 2. Канал/чат + подписка

Для привязки продукта к Telegram-каналу или группе:

1. Создайте **приватный канал** или **супергруппу**
2. Добавьте бота как администратора с правом **«Блокировать пользователей»**
3. В настройках продукта в поле **«Ссылки на чат/канал»** вставьте ссылки (каждую с новой строки):
   - `https://t.me/channel` — публичный канал
   - `https://t.me/joinchat/xxx` или `https://t.me/+xxx` — инвайт на приватный чат/канал
   - `-1001234567890` — числовой ID (если ссылку получить нельзя)
4. При оплате пользователь получает ссылки и **кнопки «Канал» / «Чат»** прямо в оплаченной секции
5. При истечении подписки — бот исключает его из всех указанных чатов/каналов
6. После продления — доступ возвращается

---

## 3. Напоминания об истечении (3, 2, 1 день)

Для продуктов с `billingType: TEMPORARY` и `durationDays`:

- За 3 дня — сообщение «Подписка истекает через 3 дня. Продлите, чтобы остаться в канале»
- За 2 дня — аналогичное
- За 1 день — аналогичное
- В день истечения — исключение из канала (если продукт привязан к чату)

---

## 4. Требования к боту

- Бот должен быть **администратором** каждого канала/группы
- Нужно право **«Блокировать пользователей»** (ban users)
- Ссылки: проще всего вставить `https://t.me/...` — система сама распарсит
